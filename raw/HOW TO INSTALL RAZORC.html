Installing RazorC is pretty simple there are two ways to download and install razorC. First we will discuss the easiest way. 

 

Method 1

First download Microsoft WebMatrix install the program. Once you have installed the program click on New > App Gallery, then on the left click on CMS look for RazorC. Click next and razorC should be installed. You can also use the search bar at the top. Note: razorC does not require a database connection to SQL Server it uses SQL Server Compact edition so it's basically like a portable database. (Something like Microsoft Access database.).

 

Method 2

Downlaod razorC from Codeplex extract the zip file and put it in a directory where you can find (my documents probably best). Then open WebMatrix click on Open> Folder. Select the razorC folder click OK and there you have installed it!.
